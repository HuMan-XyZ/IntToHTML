- HTML was created by Tim Berners-Lee in 1995.
- HTML stands for HyperText Markup Language.
- In HTML, we use tags to code web pages. Each tag must start and end with "<" and ">". Example: <p>
- There are two types of tags. They are Container tags and Empty tags.
- A empty tag will just have one tag whereas in a container tag, you have the main tag and a closing tag which is the main tag with a backslash(/) before it. Example: <html>, </html>

BASIC TAGS

1. <html> - This is the most important tag in all of HTML. Without this tag, the web page will not work and appear. <html> tag is a container tag and is put at the start and end of the html document.

2. <head> - This is one of the two essential container tags, the other being the <body> tag. In the <head> tag, we can use the <title> tag to name the webpage, and we can use others for other advanced purposes.

3. <title> - Using the <title> tag, we can put a name for the website. Here, we have used <title>Introduction to HTML</title>

4. <body> - This tag is the other essential container tag. The <body> tag contains all the other elements of the webpage. Even this text you are reading right now is inside the <body> tag.

5. <hN> - This is the header tag. The header tag has 6 levels, with <h1> being the biggest and <h6> being the smallest.
Here is an example:

<h1>

<h2>

<h3>

<h4>

<h5>
<h6>


6. <p> - The <p> or paragraph tag is a tag that slightly increases and formats normal text and makes them into paragraphs. These 6 points are also under a paragraph tag.

COOL TEXT TRICKS - ALL ARE CONTAINER TAGS

1. <b> - This is the bold tag. Any text that comes under this tag will become bold. Example: Boldddd.

2. <i> - This is the itacilise tag. Any text that comes under this tag will get italicised. Example: Leaning Tower Of Pisa.

3. <u> - This is the underline tag. Any text that comes under this tag will have a line under the text. Example: Learn to underline keywords in science.

4. <s> - This is the strike-through tag. Any text that comes under this tag will get striked through/ have a line in it. Example: We all make scratches while writing.

5. <progress> - In the progress tag, the values given for value and max turn into a progress bar. Example:

6. <kbd> - This tag is a shortform for a keyboard tag. When text is put under this tag, it changes and looks like a keyboard key. Even the tags in the start like <kbd> are <kbd> tags.

3. <div> - This tag is a container tag. It represents all tags under it as a section. It can be independently changed. It arranges things in a vertical format.
